STEP 1: Restart Node.js Server (Ubuntu)
────────────────────────────────────────
  $ pkill -f "node server.js"
  $ node server.js

  Expected: 🚀 Server running at http://0.0.0.0:3001


STEP 2: Configure Firewall (Ubuntu)
────────────────────────────────────
  $ sudo ufw allow 3001/tcp
  $ sudo ufw reload


STEP 3: Access from Windows Browser
────────────────────────────────────
  1. Get Ubuntu IP: $ hostname -I
  2. Open browser on Windows
  3. Go to: http://192.168.10.50:3001
     (Replace 192.168.10.50 with your Ubuntu IP)
