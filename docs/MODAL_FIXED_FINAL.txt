HOW TO USE:
═════════════════════════════════════════════════════════════════════════════

1. Click on an employee to open training overview modal
2. The modal shows all their training records in a clean table
3. Click on a training row to select it (row highlights in blue)
4. Use the footer buttons:

   ➕ Add Training
   • Opens form to add new training
   • Pre-selects current employee
   • Click Save to add

   ✏️ Edit
   • Opens form to edit selected training
   • Shows current training data
   • Click Save to update

   🗑️ Delete
   • Deletes selected training
   • Shows confirmation dialog
   • Refreshes modal after delete

   🖨️ Print
   • Opens print window
   • Shows formatted employee training record
   • Auto-triggers print dialog
   • Closes after printing

   Close
   • Closes the modal
   • Returns to employee list
