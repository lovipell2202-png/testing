SCRIPT LOADING ORDER (IMPORTANT):
═════════════════════════════════════════════════════════════════════════════

The scripts MUST load in this order:

1. ui-helpers.js
   └─ Provides: window.UIHelpers

2. dropdown-manager.js
   └─ Provides: window.DropdownManager
   └─ Depends on: (none)

3. data-loader.js
   └─ Provides: window.DataLoader
   └─ Depends on: (none)

4. employee-manager.js
   └─ Provides: window.EmployeeManager
   └─ Depends on: UIHelpers, DataLoader

5. training-manager.js
   └─ Provides: window.TrainingManager
   └─ Depends on: UIHelpers, DropdownManager, DataLoader, EmployeeManager

6. app-refactored.js
   └─ Initializes app
   └─ Exposes all functions to window
   └─ Depends on: All modules above

7. Page-specific scripts (all-employees.js, training-records.js, etc.)
   └─ Uses functions from modules
   └─ Depends on: All modules above
