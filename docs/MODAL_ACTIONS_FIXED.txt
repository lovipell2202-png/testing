HOW TO USE:
═════════════════════════════════════════════════════════════════════════════

1. Click on an employee to view their training overview
2. The modal opens showing all their training records
3. Click on a training row to select it (row highlights)
4. Use the footer buttons:
   • ➕ Add Training - Add new training for this employee
   • ✏️ Edit - Edit the selected training
   • 🗑️ Delete - Delete the selected training
   • 🖨️ Print - Print the overview
   • Close - Close the modal

═════════════════════════════════════════════════════════════════════════════

BUTTON FUNCTIONALITY:
═════════════════════════════════════════════════════════════════════════════

Add Training (➕)
  • Opens training form modal
  • Pre-selects current employee
  • Allows adding new training record

Edit (✏️)
  • Requires selecting a training row first
  • Opens edit form with training data
  • Allows modifying training details

Delete (🗑️)
  • Requires selecting a training row first
  • Shows confirmation dialog
  • Deletes training record from database
  • Refreshes the overview modal

Print (🖨️)
  • Prints the employee training overview
  • Shows print dialog
  • Includes employee info and all trainings

Close
  • Closes the modal
  • Returns to previous view
