📁 DATABASE TABLES (Created in NSB_Training)
═══════════════════════════════════════════════════════════════════════════════

1. Users Table
   Columns:
   - id (BIGINT) - Primary Key
   - employee_id (BIGINT) - Foreign Key to Employees
   - username (NVARCHAR) - Unique
   - password_hash (NVARCHAR) - SHA256 hash
   - role (NVARCHAR) - admin, manager, employee
   - is_active (BIT) - Account status
   - created_at (DATETIME)
   - updated_at (DATETIME)
   
   Indexes:
   - idx_users_username
   - idx_users_employee_id
   - idx_users_role

2. Sessions Table
   Columns:
   - id (BIGINT) - Primary Key
   - user_id (BIGINT) - Foreign Key to Users
   - session_token (NVARCHAR) - Unique token
   - ip_address (NVARCHAR)
   - user_agent (NVARCHAR)
   - created_at (DATETIME)
   - expires_at (DATETIME)
   
   Indexes:
   - idx_sessions_user_id
   - idx_sessions_token
   - idx_sessions_expires_at
