WHAT GETS ADDED
================================================================================

Five new columns to ExamQuestions table:

1. identification_items_json (VARCHAR MAX)
   - Stores items as JSON array
   - Example: [{"id":1,"text":"Item","answer":"A"}]

2. identification_image_url (VARCHAR MAX)
   - Stores image file path
   - Example: /uploads/images/identification_1710329400000.jpg

3. identification_title (VARCHAR 200)
   - Question title
   - Example: "Engine Components"

4. identification_instruction (VARCHAR MAX)
   - Question instructions
   - Example: "Identify the following parts"

5. identification_answer (VARCHAR MAX)
   - Concatenated answers
   - Example: "A | B | C"
